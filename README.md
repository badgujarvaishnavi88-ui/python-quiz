# Quiz Application

A console-based Quiz Application developed using Python.
The application asks questions, checks user answers, and calculates the final score.

## Technologies Used

- Python

## Features

- Question and answer system
- User input handling
- Score calculation
- Performance evaluation

## Concepts Used

- Lists
- Dictionary
- Loops
- Conditional Statements
- Functions

## Project Purpose

This project demonstrates Python logic building and problem-solving skills.

## How to Run

1. Install Python
2. Run the file:

python quiz_application.py
